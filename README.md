# fintom

A fast, lightweight C/C++ library for working with finite automata. Build, manipulate, and analyze DFAs, NFAs, and regular expressions with a clean, intuitive API.

Features:

- Deterministic (DFA) and nondeterministic (NFA) finite automata
- Regex to automaton conversion
- Minimization and optimization algorithms
- State reachability and equivalence checking
- Zero dependencies,
